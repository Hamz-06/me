Tank game - language C#

I created a game in unity which consists of having 5 levels in which the player has to defeat the enemy tanks. The enemy tank moves, shoots and rotates all on its own through a script I created.
The player has a limited amount of bullets and armour. However he can pick up loot from the floor which spawn randomly around the map.
I also learned the fundamentals of game design and managed to create a start and end game mode.

What I learned?

  • Constructors and objects
  • Physics and maths calculations
  • Created various functions
  • Used loops and if statements
  • Input and output data
  • Artificial intelligence



    • Automated trading bot - language MQL4

    During my summer holidays I spent my time learning how to trade and analyze the markets. I decided to mix my interest with trading and computer science to form an automated trading strategy using indicators such as moving averages and relative strength index. I created a total of 4 versions in which I improve the strategy.
    MQL4 is a language that is linked to a popular trading platform called metatrader. This enabled me to see real time prices and use built in indicator functions. MQL4 has pre built libraries which make it easier to code.

    What I learned?

  • Thinking methodically
  • Passing by reference
  • Commenting code
  • Being more resilient
  • Thinking creatively
  • Boolean conditions
  • statistical analysis


    • This algorithm uses 2 hull moving averages with an exponential moving average which determines the trend of the market. The 2 hull moving averages trigger a buy or sell to be placed.
    I am currently working on adding another indicator called the relative strength index. This will add extra confirmation to trades when they are placed.
